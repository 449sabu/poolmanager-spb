## Simple Pages Builder for Cardano SPOs
- ステークプールの情報をまとめたシンプルなシングルページを簡単に公開できるサービスです。

## 技術構成
- TypeScript
- Next.js
- Chakra ui
- SWR
- Blockfrost SDK
- Mesh SDK

## Getting Started

### 使い方
 1. このリポジトリをフォークする。
 2. PoolManagerを使用するか自分でエンドポイントを設定し、公開する情報を作成する。
 3.
